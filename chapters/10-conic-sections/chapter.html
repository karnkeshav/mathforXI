Conic Sections

Circle → Parabola → Ellipse → Hyperbola

Circle Parabola Ellipse Hyperbola

Circle

Definition: Set of points equidistant from a center $(h, k)$.

Distance = Radius $r$.

Standard Equation:

$(x-h)^2 + (y-k)^2 = r^2$.

General Equation:

$x^2 + y^2 + 2gx + 2fy + c = 0$.

Center: $(-g, -f)$. Radius: $\sqrt{g^2+f^2-c}$.

Note
If center is origin (0,0), then $x^2 + y^2 = r^2$.

Find center and radius of $x^2 + y^2 - 8x + 10y - 12 = 0$:

$2g = -8 \implies g = -4$. $2f = 10 \implies f = 5$. $c = -12$.

Center: $(-g, -f) = (4, -5)$.

Radius: $\sqrt{16 + 25 - (-12)} = \sqrt{53}$.

Parabola

Definition: Locus of points equidistant from a fixed point (Focus) and a fixed line (Directrix).

$e=1$.

Standard Forms ($a>0$):

1. Right: $y^2 = 4ax$. Focus $(a, 0)$.

2. Left: $y^2 = -4ax$. Focus $(-a, 0)$.

3. Up: $x^2 = 4ay$. Focus $(0, a)$.

4. Down: $x^2 = -4ay$. Focus $(0, -a)$.

Latus Rectum:

Length of chord through focus perpendicular to axis = $4a$.

Tip
Linear variable tells the axis of symmetry.
$y^2=4ax$: Symmetry about X-axis.

Find focus of $y^2 = 12x$:

$4a = 12 \implies a = 3$.

Right handed parabola. Focus $(3, 0)$. Directrix $x = -3$.

Ellipse

Standard Equation ($a > b$):

$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$.

Major Axis: 2a (along X). Minor Axis: 2b.

Relation:

$c^2 = a^2 - b^2$ (where $c$ is distance to focus).

Foci: $(\pm c, 0)$.

Eccentricity ($e$):

$e = c/a < 1$.

Vertical Ellipse
If denominator of $y^2$ is larger, major axis is Y-axis.
Foci: $(0, \pm c)$.

Find foci of $x^2/25 + y^2/9 = 1$:

$a^2=25, b^2=9$. $c^2 = 25-9=16 \implies c=4$.

Horizontal ellipse. Foci $(\pm 4, 0)$.

Hyperbola

Standard Equation:

$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$.

Transverse Axis: 2a (along X).

Relation:

$c^2 = a^2 + b^2$.

Foci: $(\pm c, 0)$.

Eccentricity ($e$):

$e = c/a > 1$.

Conjugate Hyperbola
$\frac{y^2}{a^2} - \frac{x^2}{b^2} = 1$.
Transverse axis along Y.

Find foci of $x^2/9 - y^2/16 = 1$:

$a^2=9, b^2=16$. $c^2 = 9+16=25 \implies c=5$.

Foci $(\pm 5, 0)$.

Tips & Tricks – Conics

Identifying Conic:

Look at $x^2$ and $y^2$ terms.

Same coeffs (+): Circle.

Diff coeffs (+): Ellipse.

One minus (-): Hyperbola.

Only one squared: Parabola.

Parametric Forms:

Circle: $(r\cos\theta, r\sin\theta)$.

Parabola: $(at^2, 2at)$.

Ellipse: $(a\cos\theta, b\sin\theta)$.

Latus Rectum Lengths
Parabola: $4a$
Ellipse/Hyperbola: $2b^2/a$
