Subjective Questions

Q1. Evaluate the limit: $$\lim_{x \to 1} \frac{x^2 - 1}{x - 1}$$

Show Answer
Solution: $$\lim_{x \to 1} \frac{(x-1)(x+1)}{x-1} = \lim_{x \to 1} (x+1) = 1+1 = 2$$

Q2. Find the derivative of $f(x) = x^2$ using First Principles.

Show Answer
Solution: $$f'(x) = \lim_{h \to 0} \frac{(x+h)^2 - x^2}{h} = \lim_{h \to 0} \frac{x^2+2xh+h^2-x^2}{h} = 2x$$

Q3. (Exemplar) Evaluate: $$\lim_{\theta \to 0} \frac{\sin \theta}{\theta}$$

Show Answer
Solution: This is a standard trigonometric limit theorem. The value is $1$.

Q4. Find the derivative of $f(x) = x^3$ using First Principles.

Show Answer
Solution: $$\lim_{h \to 0} \frac{(x+h)^3 - x^3}{h} = \lim_{h \to 0} \frac{x^3+3x^2h+3xh^2+h^3-x^3}{h} = 3x^2$$

Q5. (Application) If displacement is $s = t^2$, find the velocity at time $t$.

Show Answer
Solution: $v = \frac{ds}{dt} = \frac{d}{dt}(t^2) = 2t$.
