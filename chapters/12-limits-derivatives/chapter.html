Approach (Before Formal Limits)

All Formulas (Approach)
If $f(x)=ax+b$, then as $x\\to c$, value → $ac+b$
If $f(x)=x^n$, then as $x\\to a$, value → $a^n$
Approach depends on values near the point, not at the point

1. Find value of $2x+3$ as $x\\to4$

Formula: $ax+b$

Solution: $2(4)+3=11$

2. Find value of $x^2$ as $x\\to5$

Formula: $x^n$

Solution: $5^2=25$

3. Find value of $3x^2$ as $x\\to2$

Solution: $3(2)^2=12$

4. Find value of $1/x$ as $x\\to2$

Solution: $1/2$

5. Does approach require simplification?

No. Direct substitution is allowed.

Limits – Algebraic Techniques

All Formulas (Limits)
$$\lim_{x\\to a}(f(x)\\pm g(x)) = \lim f(x) \\pm \lim g(x)$$ $$\lim_{x\\to a}f(x)g(x) = (\lim f)(\lim g)$$ $$\lim_{x\\to a}\\frac{f(x)}{g(x)} = \\frac{\lim f}{\lim g}$$ $$\lim_{x\\to a}\\frac{x^n-a^n}{x-a}=na^{n-1}$$ Rationalization: multiply numerator & denominator by conjugate

1. $\lim_{x\\to2}\\frac{x^2-4}{x-2}$

Factor → cancel → substitute → Answer = 4

2. $\lim_{x\\to3}\\frac{x^2-9}{x-3}$

Answer = 6

3. $\lim_{x\\to1}\\frac{x^3-1}{x-1}$

Factor → cancel → Answer = 3

4. $\lim_{x\\to0}\\frac{\sqrt{1+x}-1}{x}$

Rationalize → Answer = 1/2

5. $\lim_{x\\to1}\\frac{x^{10}-1}{x^5-1}$

Use power rule → Answer = 2

Derivatives

All Formulas (Derivatives)
$$\\frac{d}{dx}(x^n)=nx^{n-1}$$ $$\\frac{d}{dx}(c)=0$$ $$\\frac{d}{dx}(u+v)=u'+v'$$ $$\\frac{d}{dx}(uv)=u'v+uv'$$ $$\\frac{d}{dx}\\frac{u}{v}=\\frac{vu'-uv'}{v^2}$$ $$\\frac{d}{dx}(\sin x)=\cos x$$

1. Differentiate $x^4$ → $4x^3$

2. Differentiate $5x^2$ → $10x$

3. Differentiate $x\sin x$ → Product rule

4. Differentiate $\frac{x}{x+1}$ → Quotient rule

5. Differentiate $\sin x+\cos x$
