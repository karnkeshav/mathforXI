Introduction to Sets

Definition: A set is a well-defined collection of objects.

Examples: Rivers of India, Vowels in English alphabet.

Not Sets: "Best players in the world" (Opinion varies).

Representation:

1. Roster Form: List all elements. $\{a, e, i, o, u\}$

2. Set-Builder Form: Define by property. $\{x : x \text{ is a vowel}\}$

Symbols:

$\in$: Belongs to. $a \in V$

$\notin$: Does not belong to. $b \notin V$

Key Concepts
Set is denoted by Capital letters: $A, B, X$.
Elements by small letters: $x, y$.
Order in Roster form doesn't matter: $\{1, 2\} = \{2, 1\}$.
Repetition is ignored: $\{1, 1, 2\} = \{1, 2\}$.

Converting Forms:

1. Roster to Set-Builder: $\{1, 4, 9, 16\} \rightarrow \{x : x = n^2, n \in \mathbb{N}, 1 \le n \le 4\}$.

2. Set-Builder to Roster: $\{x : x \text{ is an integer and } -3 < x < 7\} \rightarrow \{-2, -1, 0, 1, 2, 3, 4, 5, 6\}$.

Types & Subsets

Empty Set ($\phi$): Contains no elements. e.g. $\{x : x^2 = 4, x \text{ is odd}\}$.

Finite vs Infinite:

Finite: Countable elements. $\{1, 2, 3\}$.

Infinite: Uncountable. Set of Natural numbers $\mathbb{N}$.

Subsets ($\subset$):

$A \subset B$ if every element of A is also in B.

Number of subsets of a set with $n$ elements = $2^n$.

Intervals as Subsets of $\mathbb{R}$:

$(a, b)$: Open interval $\{x : a < x < b\}$.

$[a, b]$: Closed interval $\{x : a \le x \le b\}$.

Formulas
If $A \subset B$ and $A \ne B$, A is a proper subset.
Power Set $P(A)$: Collection of all subsets.
If $n(A) = m$, then $n(P(A)) = 2^m$.

Finding Subsets:

1. Subsets of $\{1, 2\}$: $\phi, \{1\}, \{2\}, \{1, 2\}$. Total $2^2 = 4$.

2. Check Subset: Is $\{a\} \subset \{a, b, c\}$? Yes.

Venn Diagrams

Visualising Sets:

Universal Set ($U$): Represented by a rectangle.

Subsets ($A, B$): Circles inside the rectangle.

Union ($A \cup B$): Area covering both circles.

Intersection ($A \cap B$): Common overlapping area.

Disjoint Sets: Circles that do not touch. $A \cap B = \phi$.

Visualization
Use shading to represent operations.
Union: Shade everything in A and B.

Set Operations

Union ($A \cup B$): Elements in A OR B.

$\{1, 2\} \cup \{2, 3\} = \{1, 2, 3\}$.

Intersection ($A \cap B$): Elements in A AND B.

$\{1, 2\} \cap \{2, 3\} = \{2\}$.

Difference ($A - B$): Elements in A but NOT in B.

$\{1, 2\} - \{2, 3\} = \{1\}$.

Complement ($A'$): Elements in $U$ but NOT in A.

If $U=\{1,2,3\}$ and $A=\{1\}$, then $A'=\{2,3\}$.

Laws of Operations
$A \cup B = B \cup A$ (Commutative)
$(A \cup B)' = A' \cap B'$ (De Morgan's Law)
$n(A \cup B) = n(A) + n(B) - n(A \cap B)$

Practical Problems:

Example: In a group of 50, 30 like Tea, 25 like Coffee, 10 like both.

Use formula: $n(T \cup C) = n(T) + n(C) - n(T \cap C)$.

$= 30 + 25 - 10 = 45$. So 45 people like at least one.

Tips & Tricks – Sets

Remember Definitions:

  • Set must be well-defined.
  • $\phi$ is a subset of every set.
  • Set itself is a subset of itself.

Operation Clues:

  • "OR" / "At least one" $\rightarrow$ Union ($\cup$).
  • "AND" / "Both" $\rightarrow$ Intersection ($\cap$).
  • "NOT" $\rightarrow$ Complement ($'$).

De Morgan's Laws:

Break the bar, change the sign.

$\overline{A \cup B} = \overline{A} \cap \overline{B}$

Master Formula:
$n(A \cup B \cup C) = n(A) + n(B) + n(C) - n(A \cap B) - n(B \cap C) - n(A \cap C) + n(A \cap B \cap C)$
