Subjective Questions

Q1. Write the set $\{x : x \text{ is a positive integer and } x^2 < 40\}$ in Roster form.

Show Answer
Solution: The positive integers whose squares are less than 40 are 1, 2, 3, 4, 5, 6.
So, the set is $\{1, 2, 3, 4, 5, 6\}$.

Q2. Let $A = \{1, 2, 3, 4, 5, 6\}$. Insert the appropriate symbol $\in$ or $\notin$ in the blank space: $5 \dots A$.

Show Answer
Solution: Since 5 is an element of A, we use $\in$.
$5 \in A$.

Q3. If $X$ and $Y$ are two sets such that $X \cup Y$ has 50 elements, $X$ has 28 elements and $Y$ has 32 elements, how many elements does $X \cap Y$ have?

Show Answer
Solution: Formula: $n(X \cup Y) = n(X) + n(Y) - n(X \cap Y)$.
$50 = 28 + 32 - n(X \cap Y) \implies 50 = 60 - n(X \cap Y) \implies n(X \cap Y) = 10$.

Q4. Write down all the subsets of the set $\{1, 2, 3\}$.

Show Answer
Solution: $\phi, \{1\}, \{2\}, \{3\}, \{1, 2\}, \{1, 3\}, \{2, 3\}, \{1, 2, 3\}$.
