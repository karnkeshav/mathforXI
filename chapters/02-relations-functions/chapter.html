Cartesian Product

Definition: Product of two sets $A$ and $B$ is the set of all ordered pairs $(a, b)$.

$A \times B = \{(a, b) : a \in A, b \in B\}$.

Example:

$A = \{1, 2\}, B = \{3, 4\}$.

$A \times B = \{(1, 3), (1, 4), (2, 3), (2, 4)\}$.

Properties:

$A \times B \neq B \times A$ (unless $A=B$).

$n(A \times B) = n(A) \times n(B)$.

Key Concepts
Ordered Pair: $(x, y) = (a, b) \iff x=a \text{ and } y=b$.
If either A or B is null, $A \times B = \phi$.

Relations

Definition: A relation $R$ from $A$ to $B$ is a subset of $A \times B$.

It describes a relationship between the first and second element.

Visual: Arrow diagrams.

Connect $a \in A$ to $b \in B$ if $(a, b) \in R$.

Domain & Range:

Domain: Set of all first elements in R.

Range: Set of all second elements in R.

Formula
Total number of relations from A to B is $2^{n(A) \times n(B)}$.

Example: $R = \{(x, y) : y = x+1, x \in \{1, 2, 3\}\}$.

Pairs: When $x=1, y=2$. When $x=2, y=3$. When $x=3, y=4$.

$R = \{(1, 2), (2, 3), (3, 4)\}$.

Domain: $\{1, 2, 3\}$. Range: $\{2, 3, 4\}$.

Functions

Definition: A relation where every input has exactly one output.

Rule: No element in Domain is left unmapped. No element maps to two outputs.

Analogy: Vending Machine.

Button 1 $\to$ Coke. Button 2 $\to$ Pepsi.

Allowed: Btn 1 & 2 both give Coke.

Not Allowed: Btn 1 gives Coke AND Pepsi.

Vertical Line Test:

If a vertical line cuts the graph at more than one point, it is NOT a function.

Notation
$f: A \to B$
$y = f(x)$
x is pre-image, y is image.

Checking Function Status:

1. $\{(2,1), (3,1), (4,2)\}$: Function. (Repeated y is okay).

2. $\{(2,1), (2,3), (4,2)\}$: Not a Function. (2 maps to 1 and 3).

Checkpoints

1. Every function is a relation.

2. One input can have multiple outputs in a function.

Algebra of Functions

Operations:

$(f+g)(x) = f(x) + g(x)$

$(f-g)(x) = f(x) - g(x)$

$(fg)(x) = f(x)g(x)$

$(f/g)(x) = f(x)/g(x), g(x) \neq 0$

Standard Functions:

Identity: $f(x)=x$

Constant: $f(x)=c$

Modulus: $f(x)=|x|$

Signum, Greatest Integer.

Domain Rules
$\sqrt{f(x)} \implies f(x) \ge 0$
$1/f(x) \implies f(x) \ne 0$

Finding Domain:

1. $f(x) = \sqrt{x-2}$: Condition $x-2 \ge 0 \implies x \ge 2$. Domain: $[2, \infty)$.

2. $f(x) = \frac{1}{x-3}$: Condition $x-3 \ne 0 \implies x \ne 3$. Domain: $\mathbb{R} - \{3\}$.

Tips & Tricks – Relations & Functions

Relation vs Function:

  • Relation: Any pairing.
  • Function: Faithful pairing (One input $\to$ One output).

Finding Domain:

  • Look for square roots (inside $\ge 0$).
  • Look for denominators (bottom $\ne 0$).

Finding Range:

  • Put $y = f(x)$ and solve for $x$ in terms of $y$.
  • Apply conditions on $y$.
Modulus Trick
$\sqrt{x^2} = |x|$
Domain of polynomial functions is usually $\mathbb{R}$.
