Subjective Questions

Q1. If $(x+1, y-2) = (3, 1)$, find the values of $x$ and $y$.

Show Answer
Solution: Equating components: $x+1=3 \implies x=2$ and $y-2=1 \implies y=3$.

Q2. Let $A = \{1, 2\}, B = \{3, 4\}$. Write $A \times B$. How many subsets will $A \times B$ have?

Show Answer
Solution: $A \times B = \{(1, 3), (1, 4), (2, 3), (2, 4)\}$.
Since $n(A \times B) = 4$, number of subsets = $2^4 = 16$.

Q3. Find the domain and range of the relation $R = \{(x, x+5) : x \in \{0, 1, 2, 3, 4, 5\}\}$.

Show Answer
Solution: Domain = $\{0, 1, 2, 3, 4, 5\}$. Range = $\{5, 6, 7, 8, 9, 10\}$.

Q4. Find the domain of the function $f(x) = \frac{x^2+3x+5}{x^2-5x+4}$.

Show Answer
Solution: Denominator $\neq 0$. $x^2-5x+4 \neq 0 \implies (x-1)(x-4) \neq 0$.
Domain: $\mathbb{R} - \{1, 4\}$.
