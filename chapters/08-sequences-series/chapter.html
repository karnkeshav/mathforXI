Arithmetic Progression (A.P.)

Definition: Sequence where difference between consecutive terms is constant ($d$).

$a, a+d, a+2d, \dots$

General Term ($a_n$):

$a_n = a + (n-1)d$.

Sum of n terms ($S_n$):

$S_n = \frac{n}{2}[2a + (n-1)d]$.

$S_n = \frac{n}{2}[a + l]$ (where $l$ is last term).

Property
If $a, b, c$ are in A.P., then $2b = a+c$.

Find 10th term of 2, 7, 12...

$a=2, d=5, n=10$.

$a_{10} = 2 + (10-1)5 = 2 + 45 = 47$.

Geometric Progression (G.P.)

Definition: Sequence where ratio of consecutive terms is constant ($r$).

$a, ar, ar^2, \dots$

General Term ($a_n$):

$a_n = ar^{n-1}$.

Sum of n terms ($S_n$):

If $r > 1$: $S_n = \frac{a(r^n-1)}{r-1}$.

If $r < 1$: $S_n = \frac{a(1-r^n)}{1-r}$.

Property
If $a, b, c$ are in G.P., then $b^2 = ac$.

Find 5th term of 3, 6, 12...

$a=3, r=2, n=5$.

$a_5 = 3(2)^{5-1} = 3(16) = 48$.

Relationship Between Means

Arithmetic Mean (A.M.):

$A = \frac{a+b}{2}$.

Geometric Mean (G.M.):

$G = \sqrt{ab}$.

Relation:

$A \ge G$.

(Equality holds if $a=b$).

Proof
$(\sqrt{a}-\sqrt{b})^2 \ge 0 \implies a+b-2\sqrt{ab} \ge 0 \implies \frac{a+b}{2} \ge \sqrt{ab}$.

Special Series

Sum of First n Natural Numbers ($\sum n$):

$\frac{n(n+1)}{2}$.

Sum of Squares ($\sum n^2$):

$\frac{n(n+1)(2n+1)}{6}$.

Sum of Cubes ($\sum n^3$):

$[\frac{n(n+1)}{2}]^2$.

Usage
Use these formulas to find sums of series like $1\cdot2 + 2\cdot3 + \dots$ by writing general term.

Tips & Tricks – Sequences

Selection of Terms:

3 terms in A.P.: $a-d, a, a+d$.

3 terms in G.P.: $a/r, a, ar$.

Infinite G.P.:

If $|r| < 1$, sum to infinity $S_\infty = \frac{a}{1-r}$.

Sigma Properties:

$\sum (a_k + b_k) = \sum a_k + \sum b_k$.

$\sum c = nc$.

Shortcut
To check A.P., see if $2b = a+c$.
To check G.P., see if $b^2 = ac$.
