The Imaginary Unit

Definition: $i = \sqrt{-1}$.

A complex number $z = a + ib$, where $a$ is real part, $b$ is imaginary part.

Powers of $i$:

$i^1 = i$

$i^2 = -1$

$i^3 = -i$

$i^4 = 1$

Pattern repeats every 4 powers.

Key Concepts
$Re(z) = a, Im(z) = b$.
Two complex numbers are equal if their real and imaginary parts are separately equal.

Simplifying Powers:

1. $i^{257}$: Divide 257 by 4. Remainder is 1. So $i^{257} = i^1 = i$.

2. Solving $4x + i(3x-y) = 3 + i(-6)$: Equate real: $4x=3 \implies x=3/4$. Equate imaginary: $3x-y = -6$. Substitute $x$ to find $y$.

Algebra of Complex Numbers

Addition/Subtraction:

Add real to real, imaginary to imaginary.

$(a+ib) + (c+id) = (a+c) + i(b+d)$.

Multiplication:

Use foil method and $i^2 = -1$.

$(a+ib)(c+id) = (ac-bd) + i(ad+bc)$.

Conjugate ($\bar{z}$):

If $z = a+ib$, then $\bar{z} = a-ib$.

$z\bar{z} = a^2 + b^2$.

Modulus ($|z|$):

Distance from origin. $|z| = \sqrt{a^2+b^2}$.

Inverse
Multiplicative Inverse $z^{-1} = \frac{\bar{z}}{|z|^2}$.

Division:

1. $\frac{1}{1+i}$: Multiply top and bottom by conjugate $1-i$.

Numerator: $1-i$. Denominator: $(1+i)(1-i) = 1^2 - i^2 = 1 - (-1) = 2$.

Result: $\frac{1}{2} - \frac{1}{2}i$.

Argand Plane & Polar Form

Argand Plane:

X-axis: Real axis. Y-axis: Imaginary axis.

Point $P(a,b)$ represents $z = a+ib$.

Polar Representation:

$z = r(\cos \theta + i\sin \theta)$.

$r = |z|$ (Modulus).

$\theta = \arg(z)$ (Argument).

Finding Argument:

Let $\tan \alpha = |b/a|$. Find quadrant.

I: $\theta = \alpha$. II: $\theta = \pi - \alpha$.

III: $\theta = -(\pi - \alpha)$ or $\pi + \alpha$. IV: $\theta = -\alpha$.

Formula
$x = r\cos\theta$
$y = r\sin\theta$

Convert to Polar: $1 + i\sqrt{3}$

$r = \sqrt{1^2 + (\sqrt{3})^2} = \sqrt{4} = 2$.

$\tan \alpha = |\sqrt{3}/1| = \sqrt{3} \implies \alpha = \pi/3$.

Point $(1, \sqrt{3})$ is in I Quad. So $\theta = \pi/3$.

Result: $2(\cos \frac{\pi}{3} + i\sin \frac{\pi}{3})$.

Quadratic Equations

Problem: $ax^2 + bx + c = 0$ when Discriminant $D < 0$.

Solution:

$x = \frac{-b \pm \sqrt{D}}{2a}$.

If $D = -k$ (where $k>0$), then $\sqrt{D} = \sqrt{-k} = i\sqrt{k}$.

Formula
Roots come in conjugate pairs for real coefficients.
$x = \frac{-b \pm i\sqrt{|D|}}{2a}$

Solve $x^2 + 2 = 0$:

$x^2 = -2$.

$x = \pm \sqrt{-2} = \pm i\sqrt{2}$.

Solve $x^2 + x + 1 = 0$:

$D = 1^2 - 4(1)(1) = -3$.

$x = \frac{-1 \pm \sqrt{-3}}{2} = \frac{-1 \pm i\sqrt{3}}{2}$.

Tips & Tricks – Complex Numbers

Modulus Properties:

$|z_1 z_2| = |z_1| |z_2|$

$|\frac{z_1}{z_2}| = \frac{|z_1|}{|z_2|}$

Conjugate Properties:

$\overline{z_1 + z_2} = \bar{z}_1 + \bar{z}_2$

$\overline{z_1 z_2} = \bar{z}_1 \bar{z}_2$

Argument Tricks:

Draw the point to find the correct quadrant.

Don't blindly rely on $\tan^{-1}(y/x)$.

Shortcut
Multiplicative inverse of $z$: Just find conjugate and divide by modulus squared.
